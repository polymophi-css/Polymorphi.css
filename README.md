# Polymorphi.css

Audio-first CSS framework with 41,000+ design tokens. Zero dependencies.

## Author

**Jason Wallace**

## Installation

```html
<link rel="stylesheet" href="foundation/all.foundation.css">
<link rel="stylesheet" href="tokens/all.tokens.css">
<link rel="stylesheet" href="components/all.components.css">
```

## Stats

- **189 files**
- **54,054 lines**
- **41,223 CSS custom properties**
- **2.82 MB uncompressed / 563 KB compressed**

## Layers

| Layer | Purpose | Files |
|-------|---------|-------|
| 0 - Foundation | Reset, normalize, responsive, utilities | 5 |
| 1 - Tokens | CSS custom properties (colors, typography, spacing, animation) | 48 |
| 2 - Primitives | JSON companion files for tooling | 126 |
| 3 - Components | Audio (mixer, transport) + UI (buttons, forms, cards) | 10 |
